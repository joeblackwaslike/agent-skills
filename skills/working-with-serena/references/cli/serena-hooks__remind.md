Usage: serena-hooks remind [OPTIONS]

  Set this as hook at PreToolUse to remind the agent to use Serena's tools
  instead of overrelying on read_file and grep

Options:
  --client [claude-code|vscode|codex]
                                  The client application that triggered the
                                  hook.  [default: claude-code]
  --help                          Show this message and exit.
