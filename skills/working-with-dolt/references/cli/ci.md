Valid commands for dolt ci are
                init - Creates database tables used to store continuous integration configuration
             destroy - Drops all database tables used to store continuous integration configuration
              import - Import a Dolt continuous integration workflow file into the database
              export - Export a Dolt continuous integration workflow by name
                  ls - List Dolt continuous integration workflows
              remove - Removes a Dolt continuous integration workflow by name
                view - View details of a specific Dolt CI workflow
                 run - Run a Dolt CI workflow
