NAME
	dolt table rm - Removes table(s) from the working set of tables.

SYNOPSIS
	dolt table rm <table>...

DESCRIPTION
	dolt table rm removes table(s) from the working set.  These changes can be staged using dolt add and committed using
	dolt commit

OPTIONS
	<table>
	  The table to remove
