NAME
	dolt blame - Show what revision and author last modified each row of a table

SYNOPSIS
	dolt blame [<rev>] <tablename>

DESCRIPTION
	Annotates each row in the given table with information from the revision which last modified the row. Optionally, start
	annotating from the given revision.
