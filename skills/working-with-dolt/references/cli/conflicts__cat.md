NAME
	dolt conflicts cat - print conflicts

SYNOPSIS
	dolt conflicts cat <table>...

DESCRIPTION
	The dolt conflicts cat command reads table conflicts from the working set and writes them to the standard output.

OPTIONS
	<table>
	  List of tables to be printed. '.' can be used to print conflicts for all tables.
