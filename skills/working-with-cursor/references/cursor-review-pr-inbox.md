# Pull Request Inbox

Use the Pull Request (PR) Inbox to keep track of pull requests that need your attention. Cursor Review groups pull requests into sections based on their current state, and you can create custom sections for your own workflow.

## Default repositories

Your default repositories determine which pull requests appear in the PR Inbox.

On the Free tier, you can select up to three default repositories. On the Team and Enterprise tiers, you can select up to 30.

> You can still open pull requests from repositories that are not in your default repositories. They just do not appear in your inbox.

## Default sections

Cursor Review creates seven sections by default:

- Needs your review
- Returned to you
- Approved
- Waiting for reviewers
- Drafts
- Merging and recently merged
- Waiting for author

Each section can be sorted by:

- Title
- Size of change
- Updated
- Created

## Add a section

To create a custom section, click `+` in the inbox filter. You can choose which repositories to include, add filters, and decide whether the section contributes to the inbox badge count.

## Edit a section

To update an existing section, click the settings icon in the section header. From there, you can rename the section, change its repositories, update its filters, manage its badge count, or delete it.

## Reorder sections

Drag sections to reorder them in your inbox.

## Search problems

Search for pull requests across your synced repositories directly in the app. Fuzzy search works across PR titles, descriptions, authors, and other metadata.

Open the search bar from the sidebar, or press `Cmd + K`.

## Share filters

To share a section's filters, open the section settings and click **Share filters**.

You can either share the filters as-is or generate a link that replaces your GitHub handle with the recipient's handle.

When someone opens the link, Cursor Review creates a new section from that filter configuration. If you later update the original section, the shared link does not update automatically. Generate a new link to share the latest version.
